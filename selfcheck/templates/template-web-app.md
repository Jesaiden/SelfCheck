# 模板：Web 应用 / 管理后台

## 一、重点检查项
- 登录态与权限控制
- 页面路由与刷新恢复
- 表单校验
- 列表筛选搜索分页
- 导入导出（若有）
- 浏览器兼容性
- 接口异常、空态与错误提示

## 二、默认高风险点
- 未授权访问
- 提交后数据不一致
- 刷新导致状态丢失
- 导入导出数据结构错误
- 搜索筛选条件残留
- 浏览器控制台隐藏报错

## 三、默认 P0
- 登录/进入关键页面
- 主表单提交成功
- 列表查看与详情查看
- 刷新恢复或状态提示
- 权限控制正确生效

## 四、人工补测建议
- 表单交互细节
- 浏览器兼容边角问题
- 大数据量表格体验
